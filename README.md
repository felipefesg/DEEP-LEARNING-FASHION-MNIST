# FASHION MNIST
Este projeto utiliza a base de dados Fashion MNIST, disponibilizada no TensorFlow e criada pela empresa europeia Zalando.
A base contém 70.000 imagens em tons de cinza de artigos de moda, sendo 60.000 destinadas ao treinamento e 10.000 ao teste. Cada imagem possui resolução de 28x28 pixels e está associada a uma das 10 classes de roupas e acessórios.

O objetivo do projeto é desenvolver um modelo preditivo baseado em redes neurais capazes de classificar corretamente as imagens em suas respectivas categorias. 
Para isso, são aplicadas técnicas de Deep Learning, que permitem ao modelo aprender padrões visuais presentes nos diferentes tipos de itens de moda.

O projeto demonstra as etapas essenciais do processo de modelagem, incluindo carregamento da base, preparação dos dados, construção do modelo, treinamento, avaliação e análise dos resultados. Ao final, espera-se que o modelo seja capaz de generalizar bem e oferecer um desempenho consistente na tarefa de classificação de imagens do conjunto Fashion MNIST.
